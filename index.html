커스터마이징 단계
1
기본 레이아웃 드롭인
XML 한 줄 + adView.load() 호출만으로 스크롤 가능한 캠페인 리스트가
즉시 표시됩니다.
2
브랜드 커스터마이징
재화 아이콘, 텍스트 색상, 로딩 뷰를 앱 디자인에 맞게 덮어씌웁니다. 
코드 추가 없이 renderer 설정만으로 완성할 수 있습니다.
3
완전 커스텀 레이아웃 (선택)
MCNativeAdRenderer 인터페이스로 자유로운 카드 UI가 구현됩니다.
SDK는 데이터·트래킹만 담당하며, UI 전권은 귀사에 있습니다.
4
데이터 활용
MCCampaign 객체로 캠페인·리워드·프로모션 등 실시간 데이터에 접근 가능합니다.
진행 중 배지 또는 프로모션 배지 등이 자동으로 표시됩니다.
